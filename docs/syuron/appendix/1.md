# 付録A アニールプロトン交換導波路の数値モデリング
<!-- calc_process.min.mdの内容を詳細に記述 -->

本付録では、第2章で述べたアニールプロトン交換 (APE) 導波路の作製プロセスおよびその屈折率分布の数値シミュレーション手法について詳述する。

## A.1 プロトン濃度の時間発展

APE導波路の形成過程は、初期のプロトン交換 (PE) プロセスと、それに続くアニールプロセスの2段階に分けられる。プロトン濃度 $C(x, y, t)$ の時間発展は、拡散方程式によって記述される。

### A.1.1 初期プロトン分布 (プロトン交換プロセス)
まず、結晶表面に幅 $W$ の開口部を持つマスクを通して安息香酸などの融液中でプロトン交換を行う。この工程により形成されるプロトン濃度の初期分布 $C(x, y, t=0)$ は、深さ方向 ($y$) にステップ状の分布を持つと仮定する。

$$
C(x,y, 0) = \begin{cases} C_0 & (|x| \le W/2 \text{ かつ } 0 \le y \le d_{PE}) \\ 0 & (\text{それ以外}) \end{cases}
$$

ここで、$C_0$ は飽和プロトン濃度、$d_{PE}$ はプロトン交換層の深さである。$d_{PE}$ は拡散係数 $D_{PE}$ と交換時間 $t_{PE}$ を用いて $d_{PE} = 2\sqrt{D_{PE} t_{PE}}$ と表される。

### A.1.2 アニール過程における拡散
次に、アニール処理によるプロトンの熱拡散を計算する。この過程は、以下の2次元線形拡散方程式に支配される。

$$
\frac{\partial C}{\partial t} = \frac{\partial}{\partial x}\left(D_x \frac{\partial C}{\partial x}\right) + \frac{\partial}{\partial y}\left(D_y \frac{\partial C}{\partial y}\right)
$$

ここで、$D_x, D_y$ はそれぞれ幅方向および深さ方向の拡散係数である。境界条件として、表面 ($y=0$) でのプロトンの流出がないこと（ノイマン境界条件 $\partial C / \partial y = 0$）、および計算領域の端部で濃度が0となること（ディリクレ境界条件）を課す。

## A.2 屈折率分布の構築

求めたプロトン濃度分布 $C(x, y)$ から、導波路の屈折率分布 $n(x, y)$ を算出する。APE導波路では、異常光線に対する屈折率 $n_e$ のみが増加する。波長 $\lambda$ における屈折率分布は、基板の屈折率 $n_{\text{sub}}(\lambda)$ と、プロトン濃度に比例する屈折率変化 $\Delta n$ を用いて次式で表される。

$$
n(x,y, \lambda) = n_{\text{sub}}(\lambda) + \Delta n_0(\lambda) \frac{C(x,y)}{C_0}
$$

ここで、$\Delta n_0(\lambda)$ はプロトン交換による表面屈折率変化の最大値であり、分散を持つ。

## A.3 計算パラメータ

本研究における数値計算では、以下のパラメータを使用した。

| パラメータ | 記号 | 値 | 単位 |
| :--- | :--- | :--- | :--- |
| マスク開口幅 | $W$ | $48$ | $\mu\text{m}$ |
| プロトン交換時間 | $t_{PE}$ | $8$ | h |
| プロトン交換拡散係数 | $D_{PE}$ | $0.045$ | $\mu\text{m}^2/\text{h}$ |
| アニール時間 | $t_{anneal}$ | $90$ | h |
| 深さ方向拡散係数 | $D_y$ | $1.3$ | $\mu\text{m}^2/\text{h}$ |
| 幅方向拡散係数 | $D_x$ | $1.3/1.5$ | $\mu\text{m}^2/\text{h}$ |
| 異常光線屈折率変化 (1064 nm) | $\Delta n_0(\omega)$ | $0.012$ | - |
| 異常光線屈折率変化 (532 nm) | $\Delta n_0(2\omega)$ | $0.017$ | - |

これらのパラメータを用いて構築された屈折率分布に対し、有限要素法等のモードソルバを適用することで、導波モードの実行屈折率や界分布を計算することが可能となる。
