# 縦型擬似位相整合デバイスの用途と課題

<!-- 縦型擬似位相整合デバイス -->
前節で述べたQPM技術の中でも、強誘電体基板の自発分極を基板垂直方向（Z軸方向）に周期反転させた「縦型擬似位相整合デバイス」は、広く研究・利用されている方式の一つである。
この構造の最大の利点は、材料が持つ非線形光学定数（$d$テンソル）の最大成分を活用できる点にある。非線形光学効果には方向依存性（異方性）があり、光の振動方向（偏光）と結晶軸の組み合わせによって利用できる定数の値が異なる。
ニオブ酸リチウム（$\text{LiNbO}_3$）やタンタル酸リチウム（$\text{LiTaO}_3$）といった代表的な結晶において、最も大きな値を持つ成分が $d_{33}$ である。縦型QPM構造では、光の偏光を結晶のZ軸に合わせることで、この $d_{33}$ を最大限に利用することが可能となる。これにより、他の結晶方位や方式を用いる場合に比べて、高い波長変換効率を得ることができる。加えて、電極を用いた微細加工技術により周期構造を設計できるため、材料の透過域内であれば広範な波長域の光に対して位相整合を実現できる。

<!-- 社会実装とユースケース -->
この「高い変換効率」と「設計による波長選択の自由度」という特長により、縦型QPMデバイスは、既に実用化されているレーザー光源用途から、将来の通信・量子技術まで、幅広い分野でその重要性が認識されている。
例えば、高輝度な可視光・紫外光レーザーとして、バイオイメージングやディスプレイ、精密加工といった分野で応用されている。また、通信分野では、全光学的（All-Optical）な信号処理デバイスとしての活用が期待されており、盛んに研究開発が行われている。
さらに、近年の量子情報技術においては、量子もつれ光子対、スクイーズド光、単一光子状態の生成源としての利用が注目されている。縦型QPMデバイスの持つ高い非線形性は、高効率な量子光源の実現に有利であり、量子暗号通信や光量子計算の基盤技術として期待されている。

<!-- 問題提起 -->
しかし、均一な周期構造を持つQPMデバイスはsinc関数状の狭帯域なスペクトル応答を示すことがわかっており、この特性に由来する様々な課題が存在する。まず、温度許容幅の狭さに起因する動作の不安定性である。スペクトル許容幅が狭い場合、環境温度の変化や自己発熱によって出力が変動しやすくなる。そのため、バイオイメージングや精密加工で求められる出力安定性を確保するには、厳密な温度制御が必要となる。次に、広帯域パルス光への対応限界である。通信分野等で用いられるフェムト秒パルス光に対しては、狭帯域な特性によりスペクトルの一部しか変換できず、時間波形の広がりや歪みを生じさせる。また、自発パラメトリック下方変換（SPDC）を用いた単一光子生成では、変換効率よりも状態の純粋性が重視される。しかし、均一周期構造が持つsinc関数状の位相整合特性は、信号光とアイドラー光の結合スペクトル（JSA）にサイドローブを生じさせる。このサイドローブ成分は二光子間に不要な周波数相関をもたらし、結果として取り出される単一光子の純粋性を低下させる要因となる。さらに、350 nm帯の紫外光生成を目指したカスケード第三高調波発生 (cTHG) において、一般的に検討されるタンデム構造は設計変数が不足しているという課題がある。この制約により、到達可能な変換効率が制限され、製造難易度の高い$2\mathrm{\mu m}$オーダーの短周期構造が必要となる。

本研究では、上述した課題を解決するため、主に以下の3点に取り組む。
1. SHGデバイスの広帯域化設計
2. cTHGデバイスの高効率化・長周期化設計
3. SPDCにおけるスペクトル制御
