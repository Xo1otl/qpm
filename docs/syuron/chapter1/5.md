# 自発的パラメトリック下方変換における純粋状態生成

<!-- 自発的パラメトリック下方変換 (SPDC) -->
SPDCは、単一の光子を低エネルギーの二つの光子（シグナル光とアイドラー光）に分裂させる非線形光学過程であり、量子もつれ光子対の生成源として重要である。量子情報処理への応用において、生成される光子対の量子状態純度は極めて重要な指標となる。特に、結合スペクトル振幅 (Joint Spectral Amplitude: JSA) が周波数相関を持たず、シグナル光とアイドラー光の成分に因数分解可能であることが、高純度な単一光子状態を得るための条件となる。

<!-- 位相の問題とスペクトル制御 -->
JSAの形状は、励起光のスペクトルと位相整合関数の積で決定される。理想的な純粋状態を得るためには、位相整合関数の形状をガウス関数型に制御する必要がある。従来のDuty比変調法では、振幅分布の制御は可能であるが、同時に意図しない位相変調が生じてしまうという「位相の問題」が存在するとされてきた。

<!-- 本研究のアプローチ -->
本研究では、Duty比変調における「位相の問題」の影響を理論的に解決しガウス型スペクトルを解析的に逆設計する。
