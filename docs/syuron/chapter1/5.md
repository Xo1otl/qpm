# 自発的パラメトリック下方変換における純粋状態生成

<!-- 自発的パラメトリック下方変換 (SPDC) -->
SPDCは、単一の光子を低エネルギーの二つの光子（シグナル光とアイドラー光）に分裂させる非線形光学過程であり、量子もつれ光子対の生成源として重要である。量子情報処理への応用において、生成される光子対の量子状態純度は極めて重要な指標となる。特に、結合スペクトル振幅 (Joint Spectral Amplitude: JSA) が周波数相関を持たず、シグナル光とアイドラー光の成分に因数分解可能であることが、高純度な単一光子状態を得るための条件となる。

<!-- 位相の問題とスペクトル制御 -->
JSA の形状は、ポンプ光のスペクトル包絡と位相整合関数によって主に決まる。ポンプにはガウシアンに近いスペクトルが用いられることが多く、位相整合関数についてもガウス型に近い形状へ整えることで、シグナル光とアイドラー光への因数分解可能性を高め、高純度な単一光子状態の生成につながる。
位相整合関数を整形する方法として、準位相整合（QPM）構造に対する Duty 比（ドメイン長）変調は、位相整合関数の振幅（フーリエ成分）を設計する手法として用いられてきた。しかし一方で、ドメイン長を変える操作は有効非線形係数を一般に複素量として変化させ、振幅だけでなく位相にも影響を及ぼす。結果として、振幅（スペクトル包絡）と位相を独立に設計することは容易ではなく、detuning に伴う位相の非一様な変化が、所望のスペクトル形状（チューニングカーブ）の実現可能性を制約することが指摘されている。

<!-- 本研究のアプローチ -->
本研究では、Duty比変調における「振幅と位相を独立に制御できない問題」を理論的に解決しガウス型スペクトルを解析的に逆設計する。
