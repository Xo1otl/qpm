# 本研究の目的と構成

## 本研究の目的
本研究の目的は、強誘電体分極反転構造を用いた擬似位相整合（QPM）デバイスにおいて、数理最適化手法を導入し、従来技術をより柔軟なデバイス構造設計を行うことである。
第一に、第二高調波発生（SHG）において、従来の定周期構造の接続や位相シフト領域の挿入に代わり、数理最適化を用いた非周期構造設計を行う。これにより、従来手法では困難であった広帯域かつ平坦な効率スペクトルを有する非周期構造を設計する。
第二に、カスケード第三高調波発生（cTHG）においては、数理最適化により、従来のタンデム構造よりも高効率、あるいは製造が容易な長周期構造で同等の効率となる構造を設計する。
第三に、これらのデバイス設計を支援するため、非線形結合波方程式（CWEs）に対する高速かつ高精度な数値解法であるPicard反復法を導入し、従来のRunge-Kutta法や非枯渇近似に基づく手法の問題点を解決する。
第四に、Duty比変調法における位相ずれの問題を理論的に解決し、解析的な逆設計手法を確立することで、高純度な量子もつれ光子対生成に不可欠なガウス型スペクトル制御を実現する。

## 本論文の構成
本論文は全6章から構成される。第1章では、波長変換デバイスの背景、用途、技術的課題、および本研究の目的について概説した。第2章では、本研究の基礎となる強誘電体における非線形光学効果、結合モード理論、およびCWEsの数値計算手法について解説する。第3章以降では具体的な研究成果について詳述する。第3章では、数理最適化を用いたSHGデバイスの広帯域化手法とその評価結果について述べ、第4章では、非励起減衰近似やRK法などの従来手法を超えるSOTAな数値計算方法と、最適化構造の設計について、従来構造との比較および製造容易性の観点から論じる。第5章では、自発的パラメトリック下方変換（SPDC）におけるDuty比変調法の理論的補正と、それを用いたガウス型スペクトル制御および純粋状態生成への応用について述べる。最後に第6章において、本研究の成果を総括し、今後の展望について述べる。
