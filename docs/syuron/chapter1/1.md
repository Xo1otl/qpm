# 波長変換デバイス

<!-- 光源と応用のギャップを埋める技術：身近な例から導入 -->
現代社会において、レーザー技術は通信、医療、精密計測など多分野の基盤である。しかし、レーザー光源の発振波長は物質固有のエネルギー準位に依存するため、産業界が求める波長と既存光源が提供できる波長には乖離がある。例えば、半導体リソグラフィ等の微細加工において、i線レジストに最も適した波長である355 nm紫外光を直接発振できる安定した光源は乏しい。この課題を解決する技術が、物質の非線形光学効果を利用した波長変換デバイスである。

<!-- 非線形光学の歴史と「位相不整合」という課題 -->
波長変換の基礎となる非線形光学は、1961年のFrankenらによる第二高調波発生 (SHG) の観測に始まる。しかし、実用化においては「位相不整合 (Phase Mismatch)」が技術的課題となった。当初は結晶角度を調整する複屈折位相整合 (BPM) が用いられたが、適用可能な材料や波長域に制約があった。

<!-- QPMデバイスの進化と量子技術への展開 -->
この課題に対し、1962年にArmstrongらが擬似位相整合 (Quasi-Phase Matching: QPM) を提唱した。1990年代には、ニオブ酸リチウム ($\text{LiNbO}_3$) などの強誘電体結晶に電界印加法を用いて分極反転構造を作製する技術が確立された。これにより、整合波長の設計自由度が向上し、変換効率の改善が可能となった。
