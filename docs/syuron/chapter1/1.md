# 波長変換デバイス

<!-- 光源と応用のギャップを埋める技術：身近な例から導入 -->
現代社会において、レーザーをはじめとする光技術は、情報通信、精密計測、医療、ディスプレイなど多岐にわたる分野の基盤である。しかし、光を生成するデバイス（レーザー光源）は、デバイスに使用される材料によって「発振できる光の波長」が限定される。
例えば、半導体製造のリソグラフィ工程では、i線レジストの感光に波長365 nmの紫外線が必要となる。しかし、この波長を直接かつ高効率に発振する実用的なレーザー材料は乏しい。波長変換デバイスは、「理想の波長」と、このような既存の光源が提供できる「現実の波長」のギャップを埋めるために、材料の「非線形光学効果」を利用して光の波長を変換する。

<!-- 非線形光学の歴史と「位相不整合」という壁 -->
波長変換の基礎となる非線形光学の歴史は、1960年のMaimanによるレーザー発明直後に遡る。1961年、Frankenらは水晶単結晶へのレーザー光入射により、入射光の2倍の周波数を持つ第二高調波発生 (Second Harmonic Generation: SHG) を初めて観測した。
しかし、実用的な変換効率の達成には、各波長での位相速度の不一致に起因する「位相不整合 (Phase Mismatch: PM)」が課題となった。物質の分散により、基本波と発生した高調波の間に位相ずれが生じる。これが結晶内での伝搬に伴って破壊的に干渉するため、効率的なエネルギー蓄積が妨げられる。当初は結晶の異方性を利用する複屈折位相整合法 (Birefringent Phase Matching: BPM) が用いられたが、利用可能な非線形光学定数や波長域に制約があった。

<!-- QPMデバイスの進化と量子技術への展開 -->
この課題の解決策として、1962年にArmstrongらは、非線形感受率の符号を空間的に周期反転させ、位相ずれを累積前にリセットして効率を高める擬似位相整合 (Quasi-Phase Matching: QPM) 法を提案した。
1990年代には、ニオブ酸リチウム ($\text{LiNbO}_3$) やタンタル酸リチウム ($\text{LiTaO}_3$) などの強誘電体結晶に対し、外部電界で微細な分極反転構造（QPM構造）を形成する電界印加法が確立された。これらの材料は、自発分極を反転でき、かつ大きな非線形光学定数を持つため、QPMデバイスに適している。これにより、構造設計のみで透過域内の所望の波長において高い変換効率を達成するデバイスの実用化が進展した。

<!-- 量子技術への展開 -->
さらに近年、波長変換の役割は量子情報技術へと拡大している。非線形光学結晶における自発的パラメトリック下方変換 (Spontaneous Parametric Down-Conversion: SPDC) は、1つの高エネルギーなポンプ光子から、保存則を満たす2つの低エネルギーな光子（光子対）を生成する過程である。生成された光子対は、量子もつれ光源やヘラルド単一光子源として、量子情報処理の実現に向けた重要なリソースと位置づけられている。
