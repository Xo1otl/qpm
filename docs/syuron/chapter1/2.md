# 縦型擬似位相整合デバイスの用途と課題

<!-- 縦型擬似位相整合デバイスとd33の導入 -->
前節で述べたQPM技術の中でも、強誘電体基板の自発分極を基板垂直方向（Z軸方向）に周期反転させた「縦型擬似位相整合デバイス」は、現在最も広く研究・利用されている方式の一つである。
この構造の最大の利点は、材料が持つ非線形光学定数（$d$テンソル）の最大成分を活用できる点にある。非線形光学効果には方向依存性（異方性）があり、光の振動方向（偏光）と結晶軸の組み合わせによって利用できる定数の値が異なる。
ニオブ酸リチウム（$\text{LiNbO}_3$）やタンタル酸リチウム（$\text{LiTaO}_3$）といった代表的な結晶において、最も大きな値を持つ成分が $d_{33}$ である。縦型QPM構造では、光の偏光を結晶のZ軸に合わせることで、この $d_{33}$ を最大限に利用することが可能となる。これにより、他の結晶方位や方式を用いる場合に比べて、高い波長変換効率を得ることができる。加えて、電極を用いた微細加工技術により周期構造を設計できるため、原理的には材料の透過域内であれば広範な波長域の光に対して位相整合を実現できる。

<!-- 社会実装とユースケース -->
この「高い変換効率」と「設計による波長選択の自由度」という特長により、縦型QPMデバイスは、既に実用化されているレーザー光源用途から、将来の通信・量子技術まで、幅広い分野でその重要性が認識されている。
例えば、高輝度な可視光・紫外光レーザーとして、バイオイメージングやディスプレイ、精密加工といった分野で応用されている。また、通信分野では、全光学的（All-Optical）な信号処理デバイスとしての活用が期待されており、盛んに研究開発が行われている。
さらに、近年の量子情報技術においては、量子もつれ光子対やスクイーズド光の生成源としての利用が注目されている。縦型QPMデバイスが持つ高い非線形性は、高効率な量子光源を実現する上で大きな利点となり、量子暗号通信や光量子計算の基盤技術としての期待が高まっている。

<!-- 問題提起 -->
均一な周期構造を持つQPMデバイスのスペクトル応答特性は、その構造のフーリエ変換によって決定され、中心周波数付近で sinc 関数状の分布を持つ。この特性にまつわるいくつかの課題が存在する。まず、温度許容幅の狭さが挙げられる。高効率化のためにデバイス長を延伸すると、それに反比例して温度許容幅が狭くなるため、僅かな環境温度の変化や自己発熱によって出力が不安定になりやすく、安定動作には厳密な温度制御装置が不可欠となる。これは、デバイスの消費電力増大やシステム全体の大型化を招く。次に、超短パルス光の変換における制限がある。フェムト秒レーザーのような広帯域なパルス光を入力する場合、均一構造の狭い帯域幅ではスペクトル成分の一部しか変換できない。これは変換効率の低下だけでなく、パルス時間波形の広がりや歪みを引き起こすため、超高速光通信や非線形顕微鏡といったパルス特性を重視する用途への適用が困難になる。そして、量子情報応用における純粋状態生成の困難さが挙げられる。sinc 関数状のサイドローブは、生成される光子対に不要な周波数相関をもたらすため、量子計算や量子ネットワークで必須となる、純粋で識別不可能な単一光子状態を生成することが困難である。加えて、カスケード第三高調波発生（Cascaded Thired Harmonic Generation: cTHG）のような複数の非線形過程が関与する複合プロセスにおいては、効率を最大化するための設計およびその検証は極めて困難である。
