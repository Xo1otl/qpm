# 縦型擬似位相整合デバイス

<!-- 縦型擬似位相整合デバイス -->
QPM技術の中でも、強誘電体基板の自発分極を基板垂直方向（Z軸方向）に周期反転させた「縦型擬似位相整合デバイス」は、広く研究・利用されている方式の一つである。
この構造の最大の利点は、材料が持つ非線形光学定数（$d$テンソル）の最大成分を活用できる点にある。非線形光学効果には方向依存性（異方性）があり、光の振動方向（偏光）と結晶軸の組み合わせによって利用できる定数の値が異なる。
ニオブ酸リチウム（$\text{LiNbO}_3$）やタンタル酸リチウム（$\text{LiTaO}_3$）といった代表的な結晶において、最も大きな値を持つ成分が $d_{33}$ である。縦型QPM構造では、光の偏光を結晶のZ軸に合わせることで、この $d_{33}$ を最大限に利用することが可能となる。これにより、他の結晶方位や方式を用いる場合に比べて、高い波長変換効率を得ることができる。加えて、電極を用いた微細加工技術により周期構造を設計できるため、材料の透過域内であれば広範な波長域の光に対して位相整合を実現できる。

<!-- 社会実装とユースケース -->
この「高い変換効率」と「設計による波長選択の自由度」という特長により、縦型QPMデバイスは、既に実用化されているレーザー光源用途から、将来の通信・量子技術まで、幅広い分野でその重要性が認識されている。例えば、高輝度な可視光・紫外光レーザーとして、バイオイメージングやディスプレイ、精密加工といった分野で応用されている。また、通信分野では、全光学的（All-Optical）な信号処理デバイスとしての活用が期待されており、盛んに研究開発が行われている。さらに、近年の量子情報技術においては、量子もつれ光子対、スクイーズド光、単一光子状態の生成源としての利用が注目されている。縦型QPMデバイスの持つ高い非線形性は、高効率な量子光源の実現に有利であり、量子暗号通信や光量子計算の基盤技術として期待されている。

<!-- SHG,SFG,cTHG,SPDCの紹介 -->
利用可能な励起光源の波長と目的とする出力光の波長の組み合わせにより、縦型QPMを用いた多様なスキームが存在する。本研究においては、擬似位相整合構造を用いた波長変換の中でも、特に**第二高調波発生（Second-Harmonic Generation: SHG）**、**和周波発生（Sum-Frequency Generation: SFG）**、これらを連鎖的に組み合わせた**カスケード第三高調波発生（Cascaded Third-Harmonic Generation: cTHG）**、そして量子光源技術として重要な**自発的パラメトリック下方変換（Spontaneous Parametric Down-Conversion: SPDC)**に焦点を当てる。
