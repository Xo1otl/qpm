# 第二高調波発生デバイスの広帯域化設計

SHGは、非線形光学結晶に入射した基本波からその2倍の周波数（1/2の波長）を持つ光を生成する非線形光学過程である。理論的詳細については第2章で述べる。

均一な周期構造を有するQPMデバイスの位相整合特性は、波長に対してsinc関数状の応答を示し、その変換帯域幅は原理的に狭いことが知られている。この狭帯域性は、特定の単一波長を変換する用途においては高い効率を実現可能であるが、実用化の観点からは主に二つの課題が存在する。

第一に、温度許容幅の狭さに起因する動作の不安定性である。QPMデバイスの最適位相整合波長は温度依存性を持つため、わずかな環境温度の変化や高出力動作に伴う自己発熱によって位相整合条件が満たされなくなり、出力強度が大きく変動する。したがって、バイオイメージングや精密加工などの用途で求められる高い出力安定性を確保するためには、高精度かつ厳密な温度制御が不可欠となる。

第二に、広帯域光や超短パルス光への対応限界である。光通信分野などで用いられるフェムト秒パルス光は広いスペクトル幅を有するが、狭帯域なQPMデバイスではスペクトル成分の一部しか波長変換に寄与しない。これにより、変換効率が制限されるだけでなく、変換後のパルスにおいて時間波形の広がりや歪みなどの劣化が生じる問題がある。

以上のことから、温度変動などの外乱に対してロバストであり、かつ広帯域な光源にも適用可能な、平坦で広い帯域特性を持つQPMデバイスの実現が強く求められている。

<!-- 先行研究手法とその課題 -->
広帯域化を実現するために、従来より主に二つのアプローチが検討されてきた。第一のアプローチは、分極反転周期を素子長方向に連続的に変化させるチャープ構造である。この手法は、素子内の異なる位置で異なる位相整合条件を満たすことで広帯域性を得るが、単位長さあたりの寄与が分散するため変換効率が低減するという難点がある。また、スペクトル上に生じるリップルの制御が困難であることも実用上の課題である。第二のアプローチは、複数の周期構造を直列に連結し、接続部の位相シフトにより調整を行う手法であるが、原理的に達成可能な帯域幅に限界があった。

<!-- 本研究のアプローチ -->
本研究では、従来の経験則や単純な構造変調に代わり、数理最適化を用いることで、広帯域化と平坦なスペクトル応答の両立を目指す。具体的には、分極反転構造を構成する各ドメインの長さを独立した設計変数として扱い、目標とする周波数応答と計算結果との誤差を最小化する最適化問題を定式化する。そして最適化アルゴリズムを用いて探索することで、従来手法を上回る特性を有する不規則周期変調構造を設計する。
