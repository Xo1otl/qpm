# 第二高調波発生効率スペクトルの広帯域化

第1.2節で述べたように、均一周期構造を持つQPMデバイスにおいては、高効率化と許容帯域幅・温度許容幅の確保との間に回避困難なトレードオフが存在する。また、超短パルス光の波形歪みを抑制するためにも、十分な帯域幅の確保は不可欠である。これらの課題を解決し、実用性の高いデバイスを実現する手段として、SHGにおける変換効率スペクトルの広帯域化が重要な研究テーマとなっている。

<!-- 従来手法と課題 -->
広帯域化を実現するために、従来より主に二つのアプローチが検討されてきた。第一のアプローチは、分極反転周期を素子長方向に連続的に変化させるチャープ構造である。この手法は、素子内の異なる位置で異なる位相整合条件を満たすことで広帯域性を得るが、単位長さあたりの寄与が分散するため変換効率が低減するという難点がある。また、スペクトル上に生じるリップルの制御が困難であることも実用上の課題である。第二のアプローチは、複数の周期構造を直列に連結し、接続部の位相シフトにより調整を行う手法であるが、原理的に達成可能な帯域幅に限界があった。

<!-- 本研究のアプローチ -->
本研究では、従来の経験則や単純な構造変調に代わり、数理最適化を用いることで、広帯域化と平坦なスペクトル応答の両立を目指す。具体的には、分極反転構造を構成する各ドメインの長さを独立した設計変数として扱い、目標とする周波数応答と計算結果との誤差を最小化する最適化問題を定式化する。そして最適化アルゴリズムを用いて探索することで、従来手法を上回る特性を有する不規則周期変調構造を設計する。
