# 第二高調波発生効率スペクトルの広帯域化

## 従来手法
第二高調波発生（SHG）における変換効率スペクトルの広帯域化に向けて、従来より主に二つのアプローチが検討されてきた。
第一のアプローチは、分極反転周期を素子長方向に連続的に変化させるチャープ構造（Chirped QPM）である。この手法は、素子内の異なる位置において異なる波長の位相整合条件を満たすことにより、広帯域な波長変換を実現する。しかしながら、広帯域化に伴い単位長さあたりの寄与が分散するため、変換効率が低下しやすいという欠点がある。加えて、スペクトル上に生じるリップルの制御が困難であるという課題も残されている。
第二のアプローチは、周期構造内に位相シフト領域を部分的または周期的に導入する手法である。例えば、複数の周期構造を直列に連結し、その接続部に適切な位相シフトを持たせることで、スペクトル形状を制御し帯域幅を拡大することが試みられてきた。しかし、先行研究で扱われてきたような限られたセクション数（分割数）の構成では、原理的に達成可能な帯域幅は限界が存在する。

## 本研究のアプローチ
本研究では、従来の経験則に基づいた設計や単純な構造変調に代わり、数理最適化アルゴリズムを導入することで、さらなる広帯域化と平坦なスペクトル応答の実現を目指す。具体的には、分極反転構造を構成する各ドメインの長さを独立した設計変数として扱い、所望のスペクトル形状と計算結果との誤差を最小化する最適化問題を定式化する。この問題を解くことにより、従来手法では導出が困難であった、優れた特性を有する非周期構造を設計する。
