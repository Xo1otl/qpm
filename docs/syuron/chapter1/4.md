# カスケード第三高調波発生の高効率化

## 従来手法
カスケード第三高調波発生（cTHG）は、第二高調波発生（SHG）と和周波発生（SFG）を連続的に生じさせることで第三高調波を得る手法である。従来、デバイス前半をSHG用、後半をSFG用としたタンデム構造が用いられてきた。しかし、この構造を用いて350nm帯の紫外光を発生させる場合、短い分極反転周期が必要となり、その作製は困難である。また、これらの非線形光学過程を記述する結合波方程式の数値解法としては、Runge-Kutta法などの一般的な手法が用いられてきた。

## 本研究のアプローチ
本研究では、数理最適化を用いてSHGとSFGを統合した非周期分極反転構造を設計し、変換効率の改善を図る。また、最適化計算を通じて、製造難易度の高い短周期を含む構造に代わり、同等の効率を維持しつつ作製容易な長周期のみの構造の探索も行う。一方で、cTHGの結合波方程式はSHGに比べて複雑である。数理最適化を行う場合、計算コストの高い厳密な数値計算を大量に反復するため、多大な計算時間を要することが課題となる。そこで本研究では、新たな高速計算手法を導入することで、この問題を解決する。
