# カスケード第三高調波発生デバイスの高効率化・長周期化設計

cTHGは、同一デバイス内でSHGとSFGを段階的に生じさせることで、入射した基本波の3倍の周波数（1/3の波長）を持つ光を生成する手法である。本手法を用いることで、例えば1064 nmの赤外レーザー光から355 nmの紫外光への変換が可能となる。SFGおよびcTHGの理論的詳細は第2章で述べる。

<!-- cTHGの応用先の例 -->
波長350 nm帯の紫外光は、電子基板の微細加工などに用いられるi線レジストの露光に適しており、高い需要がある。現在、この波長域のコヒーレント光源として、LBOやBBOといった複屈折位相整合材料を用いた波長変換レーザーが利用されている。これに対し、cTHGを用いたQPM光源が、小型、長寿命かつ安定した産業用光源としての応用が期待されている。

<!-- 一般的な構造とその課題 -->
一般的なcTHGデバイスは、SHG用とSFG用の周期構造を直列に配置するタンデム構造として設計される。しかし、この構造は設計の自由度が少なく、原理的に到達可能な効率に制限がある。加えて、この構造の実現には難易度の高い$2 \mathrm{\mu m}$の短周期構造が必要となる。

<!-- 本研究のアプローチ -->
本研究では、数理最適化を用いてSHGとSFGを統合的に扱い、不規則周期変調構造を探索することで、タンデム構造では達成できなかった高効率化構造、あるいは作製が容易な長周期での同等効率構造を設計する。

<!-- 理論的な課題2 -->
従来、変換効率が低い条件での波長変換シミュレーションでは、非励起減衰近似を用いることで高速な計算が行われてきた。一方で、高効率なデバイスの出力を算出するためには、励起光の減衰を考慮できるRunge-Kutta法やDOP853などの一般的な数値解法が使用されている。本研究では高効率化を目指すため、近似を用いた計算手法は適用できない。しかし、一般的なソルバは計算コストが高く、数理最適化に伴う膨大な回の反復計算に用いる場合、その処理時間がボトルネックとなり、実用的な設計が困難となる。

<!-- 本研究のアプローチ2 -->
そこで本研究では、相互作用描像における摂動論に基づいた、この問題に特化した数値計算手法を確立する。この手法は、非励起減衰近似を用いた場合と同等の速度で、励起光の減衰を考慮した結果を得ることができる。
