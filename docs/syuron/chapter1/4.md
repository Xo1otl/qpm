# カスケード第三高調波発生の高効率化

SHGにおける単一の非線形過程に対し、cTHGは、SHGおよび和周波発生 (Sum Frequency Generation: SFG) という二つの非線形過程を一つの素子内で連続的に生じさせる複合的なプロセスである。第1.2節で指摘した通り、このような複数の相互作用が競合する系においては、全体の変換効率を最大化するための構造設計が非常に困難となる。

<!-- 従来手法と課題 -->
従来、cTHGを実現する手法としては、素子の前半部をSHG用、後半部をSFG用の周期構造とし、それらを最適な相互作用長比で連結させた「タンデム構造」が広く用いられてきた。しかしタンデム構造は、二つの過程を本質的に分離して取り扱うため、結合した波長変換過程全体を統合的に最適化する設計自由度を十分に活用できない。さらに、紫外光発生を伴うSFGでは、作製限界に近い微細周期が要求される場合があり、これが製造上のボトルネックとなる。加えて、ポンプ枯渇を含む高効率領域の解析には、一般にルンゲクッタ法（Runge-Kutta: RK）などの汎用数値ソルバが用いられるが、第二章で詳述するように、高速な位相回転を含む微分方程式に対して計算効率が低く、構造探索に伴う反復計算まで考慮すると、計算時間が膨大になるという課題がある。

<!-- 本研究のアプローチ -->
本研究では、数理最適化を用いてSHGとSFGを統合的に最適化した不規則周期変調構造を設計することで、cTHGの高効率化を目指す。また、作製容易な長周期構造のみで同等の効率が達成可能な構造も設計する。また計算コストの課題に対して、相互作用描像における摂動論に基づいた高速かつ高精度な数値計算手法を確立する。これにより、複合的な非線形過程の最適化を実用的な時間で実行可能とし、従来のタンデム構造を上回る性能を持つ波長変換素子設計の実現を図る。
