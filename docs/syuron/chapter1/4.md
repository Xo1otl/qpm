# 1.4	カスケード第三高調波発生の高効率化

SHGにおける単一の非線形過程に対し、cTHGは、SHGおよび和周波発生 (Sum Frequency Generation: SFG) という二つの非線形過程を一つの素子内で連続的に生じさせる複合的なプロセスである。第1.2節で指摘した通り、このような複数の相互作用が競合する系においては、全体の変換効率を最大化するための構造設計が非常に困難となる。

<!-- 従来手法と課題 -->
従来、cTHGを実現する手法としては、素子の前半部をSHG用、後半部をSFG用の周期構造とし、それらを最適な相互作用長比で連結させた「タンデム構造」が一般的であった。しかし、タンデム構造は二つの過程を個別に扱う構成であり、結合過程全体を統合的に考慮することで、より変換効率の高い構造が期待できる。また、紫外光発生を伴うようなSFGプロセスにおいては、作製限界に近い微細な周期構造が要求されることもあり、これが製造上のボトルネックとなっている。加えて、ポンプ枯渇を伴う高効率な領域における波長変換プロセスの解析には、ルンゲクッタ法 (Runge-Kutta: RK) などの計算コストが高いソルバが用いられてきた。

<!-- 本研究のアプローチ -->
本研究では、数理最適化を導入し、SHGとSFGを統合的に最適化した不規則周期変調構造を設計することで、cTHGの高効率化を目指す。また、作製容易な長周期構造のみで同等の効率が達成可能な構造も設計する。また、計算コストの課題に対して、相互作用描像における摂動論に基づいた高速かつ高精度な数値計算手法を導入する。これにより、複合的な非線形過程の最適化を実用的な時間で実行可能とし、従来のタンデム構造を上回る性能を持つ波長変換素子の実現を図る。
