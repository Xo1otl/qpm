# 相談事項

## 1. 研究の適用範囲・ストーリー構築の可能性
### WDM (SHG+DFG) 文脈でのSHG+SFGの適用可能性
- **現状**: WDM光通信への応用を記述したいと思ったけどSHG+DFGだった、本研究の計算モデルはSHG+SFGである。
- **相談**: SHG+SFGは、WDMで必須となるSHG+DFGと物理的等価性が高い。本研究で実証したSHG+SFGのスペクトル制御手法は、WDM用デバイス設計にも直接適用可能な【カスケード過程の一般的制御手法】であると位置づける論理展開は可能？

### フェムト秒レーザー波長変換の序論への記載
- **現状**: 本研究の広帯域化技術の応用先として、フェムト秒レーザーの波長変換が考えられるが、本研究ではCW近似であり、パルス伝搬を厳密に扱っていない。
- **相談**: 広帯域SHGのユースケースとして、序論でフェムト秒レーザーの波長変換に言及可能？

## 2. 波長変換効率スペクトル制御のユースケース
### スペクトル形状制御技術の高度利用の可能性
- **現状**: 逆フーリエ変換法を用いれば、効率と引き換えに任意のスペクトル形状を高精度に設計可能。
- **相談**: 効率を引き換えにしても、精密なスペクトル制御が活きるユースケースはあるか? 量子もつれ光子対のスペクトル制御や、特定の分光計測、パルス整形など。

### 広帯域化による温度許容度向上の優位性
- **現状**: スペクトル帯域を広げることは温度変化による位相不整合に対してロバスト。しかし広帯域化の条件が判明しているのはSHGのみ。
- **相談**: 単純なSHG光源であって、熱分布による特性変動抑制が役立つ具体的なユースケースはあるか?
