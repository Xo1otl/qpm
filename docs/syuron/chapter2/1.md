# 緒言

本章では、本研究の基礎となる非線形光学の理論的背景と、波長変換特性の解析に用いられる一般的な数値計算手法について述べる。

第2.2節では、本研究で用いる強誘電体結晶$\text{LiTaO}_3$の非線形光学テンソルや屈折率分散などの材料物性と、アニールプロトン交換（APE）導波路の構造や数値モデルについて述べる。
第2.3節では、結合モード理論に基づき、非線形媒質中における光波の相互作用を記述するCWEsの定式化を行う。また、その数値解法として、一般的な常微分方程式ソルバであるRK法と、変換効率が低い領域で適用可能な非励起減衰近似に基づく解析的手法について述べる。
第2.4節では、SPDCにおける量子状態を表す波動関数と、二光子状態の周波数相関を表すJSAについて述べる。

これらの理論および計算手法は、第3章以降におけるデバイス設計および特性評価の基礎となるものである。
