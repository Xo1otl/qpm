# アニールプロトン交換導波路

## 導波路作製プロセスと特徴
本研究では、前節で述べた強誘電体結晶上に光導波路を形成する手法として、アニールプロトン交換 (Annealed Proton Exchange, APE) を用いる。プロトン交換 (PE) は、安息香酸などの弱酸の融液中に結晶を浸漬し、結晶表面付近のリチウムイオン ($\mathrm{Li}^+$) を水素イオン ($\mathrm{H}^+$、プロトン) で置換する化学プロセスである。
この置換により、異常光線に対する屈折率 ($n_e$) のみが増加し、常光線に対する屈折率 ($n_o$) は逆に減少するという特異な性質を示す。したがって、Z板結晶上に作製されたAPE導波路は、TMモード (偏光方向がZ軸に平行な光) のみを導波する単一偏光導波路となる。
この特性は、前節で議論した通り、最大の非線形光学定数 $d_{33}$ を利用するために必要な$Z$軸偏光（異常光線）のみを自然に選択・保持できるため、高効率なQPMデバイスの実現において極めて有利である。

初期のプロトン交換層は、格子定数の変化による歪みが大きく、非線形光学定数が劣化する問題があった。これに対し、プロトン交換後に熱処理（アニール）を行うことで、プロトンを結晶内部へ深く拡散させ、結晶性の回復とともに非線形光学定数を回復させることができる。
APE導波路は高い光損傷耐性を持ち、特に可視域から近赤外域の高強度光を扱う波長変換デバイスにおいて優れた特性を示す。また、低損失であり、長尺な導波路においても高い透過率を維持できる。

## 導波路の数値モデル
プロトンの熱拡散プロセスを拡散方程式でモデル化することにより、導波路内のプロトン濃度分布のシミュレーションが可能となる。本研究では、得られた濃度分布と屈折率変化の相関から屈折率分布を構築した。また、偏光の揃ったレーザ光源による基本モード励振を前提とし、この屈折率分布にモードソルバを適用することで、基本モードの電界分布を算出した。
これらの詳細な数値モデリング手法および計算パラメータについては、付録Aに詳述する。次章で議論する結合モード理論に基づくシミュレーションでは、本モデルで算出した導波モードの界分布関数および実効屈折率を用いることで、実デバイスの特性を再現している。
