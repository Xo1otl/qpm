# アニールプロトン交換導波路

<!-- 導波路作製プロセスと特徴 -->
本研究では、強誘電体結晶上にアニールプロトン交換 (Annealed Proton Exchange, APE) というプロセスで光導波路形成したデバイスを想定する。プロトン交換 (PE) は、安息香酸などの弱酸の融液中に結晶を浸漬し、結晶表面付近のリチウムイオン ($\mathrm{Li}^+$) を水素イオン ($\mathrm{H}^+$、プロトン) で置換する化学プロセスである。
この置換により、異常光線に対する屈折率 ($n_e$) のみが増加し、常光線に対する屈折率 ($n_o$) は逆に減少するという性質を示す。したがって、Z板結晶上に作製されたAPE導波路は、TMモード (偏光方向がZ軸に平行な光) のみを導波する単一偏光導波路となる。
この特性により、前節で述べた最大の非線形光学定数 $d_{33}$ を利用する$Z$軸偏光（異常光線）のみが自然に選択・保持されるため、高効率なQPMデバイスの実現に適している。

初期のプロトン交換層は、格子定数の変化による歪みが大きく、非線形光学定数の劣化を招く。これに対し、プロトン交換後に熱処理（アニール）を行うことで、プロトンを結晶内部へ深く拡散させ、結晶性の回復とともに非線形光学定数を回復させることができる。
APE導波路は高い光損傷耐性を有し、可視から近赤外域の高強度光を扱う波長変換デバイスに有用である。また、低損失であるため、長尺な導波路においても高い透過率が得られる。

<!-- 導波路の数値モデル -->
2-3以降で述べる結合係数を求めるために、プロトンの熱拡散プロセスを拡散方程式でモデル化し、導波路内のプロトン濃度分布を算出した。さらに、濃度分布と屈折率変化の相関から屈折率分布を構築し、モードソルバを適用することで基本モードの電界分布および実効屈折率を求めた。ここで、入射光は偏光の揃ったレーザ光源による基本モード励振を前提としている。
詳細な数値モデリング手法および計算パラメータは付録Aに記載する。次章の結合モード理論に基づくシミュレーションでは、ここで算出された導波モードの分布および実効屈折率を用いる。
