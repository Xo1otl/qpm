# 強誘電体非線形光学結晶

ニオブ酸リチウム ($\mathrm{LiNbO_3}$, LN) やタンタル酸リチウム ($\mathrm{LiTaO_3}$, LT) は、擬イルメナイト型の結晶構造を有し、点群 $3m$ ($C_{3v}$) に属する一軸性結晶である。
この点群は反転対称性を持たないため、二次の非線形光学効果が発現する。
結晶の対称軸 ($c$軸) 方向に自発分極 $P_s$ を有する強誘電体であり、一般にこの分極軸を$Z$軸とする直交座標系で記述される。

これらの結晶は、優れた非線形光学特性を持つだけでなく、可視から中赤外域 (約 $0.3 \sim 5 \mathrm{\mu m}$) にわたる広い透過帯域を持ち、機械的・化学的にも安定である。
良質な結晶育成技術が確立されていることとも相まって、実用的な波長変換デバイスの材料として広く用いられている。

強誘電体としての重要な性質として、抗電界 $E_c$ 以上の外部電界を印加することで自発分極の向きを反転できること (分極反転) が挙げられる。
分極反転操作は結晶格子の $180^\circ$ 回転と等価であり、これにより非線形光学定数 $d$ の符号が反転する。
この性質を利用して周期的な分極反転構造を作製することで、擬似位相整合 (QPM) が実現される。

本研究では、これらの中でも特に光損傷 (フォトリフラクティブ) 耐性に優れた $\mathrm{MgO}$ 添加定比組成タンタル酸リチウム ($\mathrm{MgO:SLT}$) を主な対象として解析を行う。
前節で述べたように、本研究では最大の非線形光学定数 $d_{33}$ を利用する $Z$ 軸偏光 (異常光線) の相互作用を扱う。
そのため、数値計算に際しては異常光線屈折率 $n_e$ の分散関係および温度依存性を考慮する。
具体的には、$\mathrm{MgO:SLT}$ の異常光線屈折率 $n_e(\lambda, T)$ に対して以下の形式のセルマイヤー (Sellmeier) 方程式を用いる。

$$
n_e^2(\lambda, T) = a_1(T) + \frac{a_2(T)}{\lambda^2 - a_3(T)^2} + \frac{a_4(T)}{\lambda^2 - a_5(T)^2} - a_6 \lambda^2
$$

ここで、$\lambda$ は波長 [$\mu\mathrm{m}$]、$T$ は温度 [$^\circ\mathrm{C}$] である。各係数 $a_i(T)$ ($i=1\dots5$) は温度依存性を持ち、基準温度 $T_0 = 24.5^\circ\mathrm{C}$ を用いて次のように定義される。

$$
a_i(T) = a_{i,0} + b_i f(T), \quad f(T) = (T - T_0)(T + T_0 + 546.32)
$$

なお、$a_6$ は温度に依存しない定数である。本研究で使用する係数セットを表 2.1 に示す。
