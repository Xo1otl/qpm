# 強誘電体非線形光学結晶

本研究では、優れた非線形光学特性を持ち、良質な結晶育成技術が確立されているニオブ酸リチウム ($\mathrm{LiNbO_3}$, LN) およびタンタル酸リチウム ($\mathrm{LiTaO_3}$, LT) を主な対象とする。特に、光損傷耐性に優れた添加物制御組成 ($\mathrm{MgO:LN}$, $\mathrm{MgO:SLT}$) は高強度波長変換デバイスにおいて重要である。

## 結晶構造と物理的性質

LNやLTは擬イルメナイト型の結晶構造を有し、点群 $3m$ ($C_{3v}$) に属する一軸性結晶である。この点群は反転対称性を持たないため、前節で述べた二次の非線形光学効果が発現する。
結晶の対称軸 ($c$軸) 方向に自発分極 $P_s$ を有する強誘電体であり、この分極軸を$Z$軸、鏡映面と垂直な方向を$Y$軸、これらに直交する方向を$X$軸とする直交座標系 ($XYZ$) で一般に記述される。
これらの結晶は、可視から中赤外域 (約 $0.3 \sim 5 \mathrm{\mu m}$) にわたる広い透過帯域を持ち、機械的・化学的にも安定であることから、実用的な波長変換デバイスの材料として広く用いられている。

## 屈折率分散とセルマイヤー方程式
光波の位相整合条件を数値計算によって正確に予測するためには、屈折率の分散関係を知る必要がある。LNやLTは負の一軸性結晶であり、常光線屈折率 $n_o$ と異常光線屈折率 $n_e$ という2つの独立した屈折率を持つ。
特に、本研究で扱う擬似位相整合 (QPM) デバイスでは、最大の非線形光学定数 $d_{33}$ を利用するために、関与する全ての光波を$Z$軸偏光 (異常光線) とする場合が多く、$n_e$ の波長依存性および温度依存性が極めて重要となる。
これらは一般にセルマイヤー (Sellmeier) 方程式によって近似され、波長 $\lambda$ と温度 $T$ の関数として与えられる。

本研究では、MgO:SLT の異常光線屈折率 $n_e(\lambda, T)$ を記述するために、以下の形式の Sellmeier 方程式を用いる。

$$
n_e^2(\lambda, T) = a_1(T) + \frac{a_2(T)}{\lambda^2 - a_3(T)^2} + \frac{a_4(T)}{\lambda^2 - a_5(T)^2} - a_6 \lambda^2
$$

ここで、$\lambda$ は波長 [$\mu\mathrm{m}$]、$T$ は温度 [$^\circ\mathrm{C}$] である。各係数 $a_i(T)$ ($i=1\dots5$) は温度依存性を持ち、基準温度 $T_0 = 24.5^\circ\mathrm{C}$ を用いて次のように定義される。

$$
a_i(T) = a_{i,0} + b_i f(T), \quad f(T) = (T - T_0)(T + T_0 + 546.32)
$$

なお、$a_6$ は温度に依存しない定数である。本研究で使用する係数セットを表 2.1 に示す。

**表 2.1: MgO:SLT の Sellmeier 係数**

| 1 | 4.5615 | $4.782 \times 10^{-7}$ |
| 2 | 0.08488 | $3.0913 \times 10^{-8}$ |
| 3 | 0.1927 | $2.7326 \times 10^{-8}$ |

## 強誘電性と分極反転
強誘電体であるこれらの結晶の最大の特徴は、抗電界 $E_c$ 以上の外部電界を印加することで、自発分極の向きを反転できることである。この現象は分極反転 (Domain Inversion) と呼ばれる。
分極反転は、結晶格子の物理的な向きを $180^\circ$ 回転させる操作と等価であり、これにより奇数階のテンソル量（自発分極 $P_s$ や非線形光学定数 $d$ など）の符号が反転する。
この性質を利用し、微細な電極パターンを用いて周期的に分極を反転させることで、相互作用する光波の実効的な位相差を補正する擬似位相整合 (QPM) が実現される。
QPMデバイスの設計においては、前節で導出した非線形分極の符号が周期的に反転することを考慮し、結合方程式を解くことになる。
