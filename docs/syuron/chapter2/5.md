# 結言

本章では、本研究の基礎となる強誘電体結晶における非線形光学相互作用の理論的枠組みと、それを解析するための数値計算手法について述べた。

第2.2節では、強誘電体単結晶LiTaO$_3$の材料特性と、アニールプロトン交換導波路の数値モデルについて概説した。
第2.3節では、結合モード理論に基づき結合波方程式を定式化し、その数値解法について議論した。一般的なRunge-Kutta法に加え、非励起減衰近似が成立する領域において、計算量をドメイン数 $N$ に対して線形オーダー $O(N)$ まで低減可能な計算アルゴリズムを示した。この手法は、不規則な分極反転構造を持つ長尺デバイスの大規模な最適化問題を扱う上で、計算コストの大幅な削減を可能にする特に有用な手法である。

第2.4節では、SPDC過程を量子力学的に記述し、生成される光子対の同時スペクトル強度（JSA）が、ポンプ光のスペクトル包絡線と、素子の非線形構造によって決定される位相整合関数の積で表されることを示した。特に、位相整合関数が非線形結合係数の空間分布のフーリエ変換に対応するという記述は、実空間での構造設計を通じて周波数空間での量子相関特性を自在に制御できることを示唆しており、これは第5章における純粋状態生成の核心となる概念である。

本章で示した理論モデルと数値解析手法は、次章以降で議論する広帯域SHGデバイスの設計、高効率cTHGデバイスの解析、およびSPDC光源の純粋状態生成において、デバイスの動作原理を理解し、その性能を定量的に評価するための基盤となる。
