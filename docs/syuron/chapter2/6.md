# 結言

本章では、本研究の基礎となる非線形光学の理論、材料物性、および数値解析手法について述べた。

2節では、強誘電体結晶である MgO:SLT の光学特性および非線形光学テンソルについて述べ、アニールプロトン交換 (APE) 導波路の特性と数値モデルについて概説した。
3節では、波長変換効率を決定づける位相不整合と、それを補償するQPMの原理について説明した。また、光波の相互作用を記述する結合波方程式を導入し、その数値解法について議論した。一般的な結合波方程式の解法としてはRunge-Kutta法等の常微分方程式ソルバが用いられるが、長尺デバイスの構造探索等の解析には計算コストが高い。これに対し、変換効率が低い領域において成立する非励起減衰近似を適用することで、計算量をドメイン数 $N$ に対して線形オーダー $O(N)$ まで低減可能な高速計算アルゴリズムを示した。また、多段の波長変換過程である cTHG における構造最適化問題が、イジングモデルの基底状態探索に帰着されることを示した。
4節では、QPM構造の設計に用いる数理最適化手法について概説した。
5節では、量子光学的な波長変換過程である自発的パラメトリック下方変換 (SPDC) について述べ、生成される二光子状態の同時スペクトル強度 (JSA) が、ポンプ光のスペクトル包絡線と位相整合関数の積で記述されることを示した。この関係性は、分極反転構造の設計を通じて位相整合関数を整形することで、生成される光子対の周波数相関特性を制御できることを示している。

本章で整備した理論モデルと数値計算手法は、次章以降で行う広帯域SHGデバイスや高純度SPDC光源の設計・解析の基盤となる。
