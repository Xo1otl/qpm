# 結言

本章では、SPDCにおける純粋状態生成を目的として、任意の位相整合関数を実現するQPMデバイスの設計手法について論じた。

特に、Duty比変調により実効的な非線形光学定数の空間分布を制御し、誤差拡散法を用いることで、作製プロセスの分解能制約（最小ドメイン幅）を満たしつつ、理想的なガウス型スペクトルを持つデバイス構造を設計可能であることを示した。

数値シミュレーションの結果、量子化された構造においても設計通りのサイドローブのないスペクトルが得られることを確認した。一方で、ガウスアポダイゼーションに伴う実効相互作用長の減少により、変換効率は一様周期構造と比較して低下することも明らかとなった。

以上の結果は、変換効率とのトレードオフは存在するものの、既存の低分解能な描画装置を用いても、高品質な量子もつれ光源や単一光子源が実現可能であることを示唆しており、量子情報通信分野への実用的な貢献が期待される。
