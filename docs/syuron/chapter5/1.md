# 緒言

本章では、量子情報通信への応用を念頭に、自発的パラメトリック下方変換（SPDC）におけるヘラルド単一光子の純粋状態生成を目的とした、擬似位相整合（QPM）デバイスの設計手法について述べる。

高品質な量子もつれ光源や単一光子源の実現には、シグナル光とアイドラ光のスペクトル相関を除去し、結合スペクトル振幅（JSA）を因数分解可能な形にする必要がある。そのためには、励起光のスペクトル形状（ガウス関数）に対し、位相整合関数も同様にサイドローブのないガウス型へと整形（アポダイゼーション）することが求められる。

本章では、QPM構造のDuty比（パルス幅）を空間的に変調することで、実効的な非線形光学定数の分布を制御し、任意の位相整合スペクトルを実現する手法を提案する。特に、作製プロセス上の制約（最小ドメイン幅や描画グリッド）を満たしつつ、理想的なガウス型スペクトルを得るために、誤差拡散法を用いた量子化アルゴリズムを導入する。最後に、数値シミュレーションにより、提案手法の有効性とトレードオフについて評価する。
