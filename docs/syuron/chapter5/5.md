# 計算と評価

前節までに述べた設計手法の有効性を検証するため、数値シミュレーションを行った。本節では、ガウス型アポダイゼーションを施した擬似位相整合デバイスのスペクトル特性と変換効率について評価する。

## シミュレーション条件

サイドローブのないガウス型の位相整合関数（スペクトル）の逆設計を行った。結合係数の空間分布 $\kappa(z)$ と位相整合関数 $\Phi(\Delta k)$ がフーリエ変換の関係にあることに基づき、$\kappa(z)$ にガウス関数型のエンベロープを適用した。

ガウス関数の標準偏差 $\sigma$ は、デバイス全長 $L$ に対して $\sigma = L/8$ と設定した。有限長デバイスによる空間分布の急激な切断は、スペクトルにサイドローブ（ギブス現象）を生じさせるが、$\sigma = L/8$ とすることで、デバイス両端（$z = \pm L/2$）における振幅は $\exp(-8) \approx 3.3 \times 10^{-4}$ まで減衰する。これにより、切断の影響を無視し、理想的なガウススペクトルを得ることが可能となる。

数値計算には、カスケードTHGと同様に、設計波長 1064 nm、温度 70 $^\circ$C における MgO:SLT のセルマイヤー方程式を用いた。構造の量子化においては、作製プロセスの分解能を考慮し、コヒーレンス長 $l_c$ （$\approx 4$ $\mu$m）を単位とする粗いグリッド上で離散化を行った。

なお、比較対象として一様周期構造を持つ従来のQPMデバイスの計算も行った。この際、一様周期デバイスの長さは、そのスペクトル（sinc関数）のメインローブの幅が、アポダイゼーションを施したデバイスのガウス型スペクトル幅と同等になるように決定した。その結果、アポダイゼーションを施したデバイスの全長は、一様周期デバイスの約3倍（$L_{\text{holo}} \approx 3 L_{\text{periodic}}$）となった。

## 結果と考察

シミュレーション結果を図 5.1 に示す。図中の黒い破線はターゲットとした理想的なガウス関数のスペクトル、緑の実線は誤差拡散法により量子化された構造（Dithered）のスペクトル、赤い実線は比較用の一様周期構造（Periodic）のスペクトルを表している。

![Simulation Result](../examples/syuron/gaussian/dithered_Lc_demo.png)
*図 5.1: 位相整合スペクトルの比較*

まず、量子化された構造のスペクトル（緑）が、ターゲットである理想的なガウス形状（黒破線）と極めて良く一致していることが確認できる。量子化ノイズの影響は見られず、ガウス関数特有のサイドローブのない滑らかなスペクトル形状が得られている。これは、誤差拡散法が低周波域の量子化誤差を効果的に除去し、設計した変調プロファイルを忠実に再現できていることを示している。

次に、変換効率に着目する。アポダイゼーションを施したデバイスのピーク効率は、その1/3の長さを持つ一様周期デバイスの効率とほぼ同等となった。非線形光学効果における変換効率は相互作用長の二乗 ($L^2$) に比例して増大するため、長さが3倍であれば本来は9倍の効率が得られるはずである。しかし、ガウス型アポダイゼーションでは、ドメインのDuty比を変化させることで実効的な結合係数を空間的に減少させている。

今回設定した $\sigma = L/8$ という条件では、効率が理論最大値（一様構造かつ全長 $L$）の約 $1/9$ に低下しており、これは実効的な相互作用長が全長の $1/3$ 程度に制限されていることと等価である。すなわち、スペクトル形状の制御（サイドローブの抑制と帯域の拡大）と引き換えに、変換効率の低下が生じていることが定量的に確認された。

以上の結果より、提案手法を用いることで、コヒーレンス長程度の粗い分解能しか持たない作製プロセスであっても、任意の位相整合関数を持つ高性能な波長変換デバイスが実現可能であることが示された。
