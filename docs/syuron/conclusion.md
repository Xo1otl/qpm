# 結論

本研究では、強誘電体を用いたQPMデバイスの高性能化を目指し、数理最適化手法および新たな数値計算法を導入することで、広帯域化、高効率化、および高純度量子状態生成におけるガウス形状位相整合関数などの高機能QPM構造設計に取り組んだ。各章で得られた成果は以下の通りである。
第1章では、波長変換デバイスの現状と課題について概説した。従来のQPM技術は、狭帯域性や設計・作製のトレードオフといった制約を抱えている。これらを克服するため、従来の経験則に代わり、数理モデルと最適化アルゴリズムを融合させた設計手法の必要性について述べた。
第2章では、非線形光学および強誘電体材料の基礎理論について述べた。MgO:SLT結晶を用いたType-0位相整合におけるSHG、SFG、cTHG過程を結合波方程式として定式化し、非励起減衰近似（UPA）を用いた高速計算手法について解説した。また、イジングモデルとの関連性、ベイズ最適化、および準ニュートン法（L-BFGS）の原理について概説するとともに、SPDCにおける純粋状態生成の理論的枠組みについて述べた。
第3章では、SHGにおける動作帯域の拡大と平坦化を実現するQPM構造の設計手法について論じた。従来のチャープ構造や位相シフト法では達成が困難とされた広帯域な平坦スペクトルと変換効率の両立を目的とし、メタヒューリスティクス探索と勾配法を組み合わせた二段階の最適化手法を提案した。目的関数に平均二乗誤差（MSE）を用いて滑らかな勾配計算を行うことで、製造可能な最小ドメイン幅の制約を満たしつつ最適化を実施した。その結果、従来の3セグメント構造と比較して約2.6倍の帯域幅を有し、かつピーク効率の80%を振幅する不規則周期変調構造が得られた。
第4章ではcTHGにおける、実用的な作製制約下での目標効率達成を目指した構造最適化と、そのための高速数値解法について述べた。振動する位相項を解析的に処理する有効結合係数と、Super-Step法やMagnus-Cayley法を導入することで、Manley-Rowe関係式を厳密に満たす高精度な計算を実現した。本手法は、構造の微細周期に依存しない粗いステップ幅での計算を可能とし、従来法と比較して大幅な高速化を達成した。この数値解法とマルチスタートL-BFGS法を用い、SFG領域に2次のQPM周期を採用したタンデム型構造を初期値として最適化を行うことで、最小ドメイン幅の制約を満たしつつ、実用的な高効率を有する不規則周期構造を見出した。また、本手法の概念をパルス伝搬解析（SSFM）へ拡張し、パルス励起下における最適化の適用可能性を示した。
第5章では、量子情報通信への応用に向けた、SPDCにおけるヘラルド単一光子の純粋状態生成のためのデバイス設計について論じた。周波数相関のない量子もつれ光源を実現するため、位相整合関数をサイドローブを持たないガウス型に整形する手法を検討した。従来のDuty比変調に加えてパルス位置を調整することによって複素実効結合係数を制御する理論モデルを構築した後、最小ドメイン幅や描画グリッドといった作製上の制約を考慮して誤差拡散法を適用した。数値シミュレーションの結果、量子化構造においても目標とするガウス型スペクトルが得られ、サイドローブが抑制されることを確認した。アポダイゼーションに伴う実効相互作用長の減少による効率低下は避けられないものの、既存のプロセスで高い純度の単一光子源が実現可能であることを示した。

以上の成果により、本研究で確立した数理最適化に基づく設計手法と高速計算アルゴリズムは、従来のQPMデバイスの限界を突破し、次世代の高効率・高性能な光量子デバイス開発における基盤技術として極めて有効であることを示した。
