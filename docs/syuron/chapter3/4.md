# 結言

本章では、第二高調波発生におけるQPM構造の広帯域化設計について検討を行った。

従来手法であるチャープ構造は、帯域幅を容易に拡大できる反面、変換効率の著しい低下やスペクトルへのリップルの発生が避けられず、矩形に近い理想的なスペクトルを得ることは困難であった。また、位相シフトを用いたセグメント構造においても、セグメント数の増加に伴いパラメータ探索空間が爆発的に増大するため、大域的な最適解を得ることは現実的ではなかった。

これに対し、本研究では各分極反転ドメインの幅を連続変数として扱い、ベイズ最適化によるターゲット形状探索と勾配法による構造最適化を組み合わせた手法を適用した。
このアプローチにより、従来の3セグメント構造と比較して約2.6倍の帯域幅を持ちながら、同時に高い変換効率（元の65%）を維持する構造を導出することに成功した。
本手法は、必要な帯域幅と平坦性を持つスペクトルを自在に設計するための強力なツールとなり得る。
