# 緒言

強誘電体結晶における分極反転構造を用いた準位相整合（QPM）デバイスは、高い波長変換効率を実現できる一方で、その許容波長帯域は一般に狭い。
この狭帯域特性は、温度変化やポンプ光の波長ドリフトに対する許容度を低下させる要因となるほか、超短パルス光のような広帯域光の波長変換においてはパルス波形の歪みや効率低下を招く。
そのため、用途に応じた適切な帯域幅とスぺクトル形状を持つ素子の設計が求められる。

本章では、第二高調波発生（SHG）における広帯域化手法について論じる。
まず、チャープ構造や位相シフト領域を用いたセグメント構造など、従来の広帯域化手法についてその特性と設計上の課題を整理する。
続いて、不規則な分極反転周期を持つ構造に対し、ベイズ最適化と勾配法（L-BFGS）を組み合わせた数理最適化手法を適用する試みについて述べる。
この手法により、従来の手法では困難であった「平坦かつ広い帯域幅」と「高い変換効率」の両立を目指す。
