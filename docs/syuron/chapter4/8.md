# 結言

本章では、強誘電体分極反転デバイスにおけるカスケード第三高調波発生（cTHG）の高効率化を目的として、結合波方程式の新たな高速数値計算法の提案と、それを用いた構造最適化について論じた。

まず、結合波方程式において、急速に振動する位相項と分極反転構造を「有効結合係数」として解析的に積分することで、振幅の緩やかな変化のみを数値的に追跡する手法を確立した。さらに、Magnus展開とCayley変換を組み合わせたスキームを導入することで、エネルギー保存則を厳密に満たしつつ、計算速度を桁違いに高速化することに成功した。

この高速計算手法を用いることで、これまで計算コストの観点から困難であった大域的な構造最適化が可能となった。作製可能な最小ドメイン幅（1.5 $\mu$m）の制約下において、SFG過程にあえて2次のQPM周期を採用するなどの物理的考察に基づいた初期値設定と、マルチスタートL-BFGS法を組み合わせることで、1W出力が得られる不規則周期変調構造を見出した。

さらに、本手法の概念をパルス伝搬解析（SSFM）へ拡張する可能性についても議論した。有効結合係数を非線形ステップに導入することで、構造の周期性に縛られない粗いサンプリングが可能となり、パルス励起下での最適化計算を現実的な時間で行える見通しを得た。

以上の成果は、QPMデバイスの設計において、計算機支援による最適化が極めて有効であることを示しており、今後の高効率波長変換デバイス開発における基盤技術となるものである。
