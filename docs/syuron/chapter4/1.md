# 緒言

本章では、カスケード第三高調波発生（cTHG）の高効率化を目指し、結合波方程式の新たな高速数値解法と構造最適化について論じる。

従来の周期的なタンデム構造を超える効率を実現するためには、不規則周期構造の最適化が不可欠である。しかし、これには膨大な計算コストがかかるため、微細なステップ幅を必要とする従来のルンゲ・クッタ法などでは困難であった。そこで本章では、有効結合係数とMagnus-Cayleyスキームを用いた高速かつ高精度な数値解法を提案する。本手法は、エネルギー保存則（ユニタリ性）を厳密に満たしつつ、計算速度を従来の約100倍に向上させる。

この高速計算手法を用い、作製制約（最小ドメイン幅 $1.5 \mu\mathrm{m}$）の下で変換効率を最大化する不規則周期構造を設計する。物理的考察に基づく初期値設定とマルチスタートL-BFGS法により、単純なタンデム構造と比較して大幅に高い効率を持つ構造が得られたことを示す。最後に、本手法のパルス伝搬解析への拡張についても議論する。
