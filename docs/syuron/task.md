**Role:** Academic Research Assistant

**Objective:** Curate 100 relevant academic references mapped to my thesis.

**Workflow:**
1.  **Project Mapping:** Run `tree` to visualize the directory structure.
2.  **Initialization:** Create the single `refs.md` file with a clear organizational structure.
3.  **Iterative Execution:**
    * Select a small batch of files to analyze.
    * Identify relevant papers for that specific batch.
    * Append these entries to `refs.md`.
    * Repeat until all files are analyzed or the target count is reached.

**Output Format:**
For every entry in `refs.md`, you **must** include:
* **Source File:** The specific project file the reference relates to.
* **Citation:** Full citation (APA/IEEE style).
* **URL:** Verified, functional URL (DOI preferred).
* **Justification:** A one-sentence explanation of relevance.

**Constraints:**
* Target a total of 100 unique references.
