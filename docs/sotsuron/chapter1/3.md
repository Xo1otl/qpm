# MgO:SLTを用いた導波路型光源の提案

前節で述べた従来光源の課題（アライメントの困難さ、結晶寿命、装置の大型化）を解決するため、本研究では新たに **MgO添加定比組成タンタル酸リチウム（MgO:SLT）を用いた導波路型擬似位相整合（QPM）デバイス** を提案する。[1]-[3]

## MgO:SLT結晶の優位性

本研究で用いる MgO:SLT は強誘電体結晶の一種であり、紫外光発生用材料として優れた特性を数多く有している [4][5]。

まず、本材料は極めて高い耐光損傷性を備えている。従来のニオブ酸リチウム（LiNbO$_3$: LN）等では、強い短波長光の入射によって「フォトリフラクティブ効果（光損傷）」と呼ばれる屈折率変化が生じ、これによるビーム品質の劣化や位相整合条件のずれが実用上の課題となっていた [6]。これに対し、MgOを添加したSLT結晶は高い耐光損傷性を有しており、導波路型光源においても光損傷の兆候が見られない安定した動作が報告されている [4][5]。また、紫外域（吸収端〜270 nm付近）に至るまで高い透明度を有していることも大きな利点である。355 nmの紫外光発生においても光吸収による自己発熱が最小限に抑えられるため、高効率な波長変換が期待できる [5]。さらに、MgO:SLTは熱伝導率が比較的高いため、レーザ照射に伴う熱を効率よく逃がすことが可能である。これにより、温度分布の不均一に起因する特性劣化を抑制し、安定した出力を維持しやすいという特徴を持つ [5]。

## 導波路構造とQPM

本提案では、上記のMgO:SLT結晶内部に光を閉じ込めるコア層となるチャネル導波路（waveguide）を作製し、さらに分極反転構造を形成することでQPM動作を実現する。[4][7][8]

バルク結晶では回折によりビームが広がるため、高い光パワー密度を維持できる距離が限られる。一方、導波路構造では数$\mu$m径のコア内に光を閉じ込めて伝搬させるため、デバイス全長（数cm）にわたって高い光パワー密度を維持できる。非線形光学効果は概ね光パワー密度の二乗に比例するため、導波路化により変換効率を大きく向上させることができる。[9][4][8]

また、導波路は光ファイバーとの結合や基板上への直接実装（ピグテール化など）が容易である。これにより、従来の空間光学系で行っていた複雑な角度調整（アライメント）が不要となり、振動に強く、小型で堅牢な光源モジュールを実現できる。[4][8]

QPMは、結晶の自発分極を周期的に反転させることで、基本波と高調波の位相速度差を補償する技術である。[7][8]  
バルク結晶の複屈折位相整合では利用できる結晶方位や非線形定数が制限されるが、QPMでは周期構造を適切に設計することで任意の波長に対して位相整合が可能となる。[7][8]

特に、MgO:SLT結晶が持つ大きな非線形光学定数 $d_{33}$（代表値として約14 pm/V）を利用できるため、一般的なバルク結晶（例：LBO等）と比較して高い変換効率が期待できる。[5][10][8]

## 2µm周期分極反転構造の作製における課題

本研究が目指す355 nm発生（THG）において、和周波発生（SFG）プロセスに用いる分極反転構造の周期は約2 $\mu$mと極めて微細である。一般に、QPMデバイスの変換効率は分極反転構造のデューティ比（反転層と非反転層の比率）の理想値（50:50）からのずれに敏感であり、微細な周期であればあるほど、電界印加時の分極壁の広がりを制御することが困難となる。[7]

特に、MgO:SLTにおいて2 $\mu$mという微細周期を高い均一性でウェハ全面に形成することは、デバイス実現における最大の技術的障壁であり、この作製プロセスの確立そのものが、高性能な紫外光源を実現するための極めて重要なマイルストーンとなる。

## まとめ

以上の特徴により、MgO:SLTを用いた導波路型QPMデバイスは、従来技術の課題であったアライメント精度や寿命の問題を根本的に緩和しつつ、小型・高効率・低コストな355 nm光源を実現する有望な候補である。[4][5][8]
