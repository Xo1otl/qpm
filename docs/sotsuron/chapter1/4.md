# 研究目的と本論文構成

## 研究目的

本研究の最終的な目標は、次世代のレーザ直接描画（LDI）装置に適用可能な、高効率・長寿命かつ小型・堅牢な波長355 nm紫外光源を実現することである。

前述したバルク結晶の課題に対し、MgO:SLT導波路におけるQPM利用は、高効率化、メンテナンスフリー化、および長期信頼性の確保を同時に達成し得る。しかし、この実現には 2 $\mu$m という極めて短い周期を持つ分極反転構造の形成が不可欠である。

そこで本研究では、**2 $\mu$m 周期分極反転構造の作製** を主な目的とする。

なお、分極反転による波長変換の理論や、なぜ 2 $\mu$m という周期が必要となるのかについては、第2章で詳述する。

## 本論文の構成

本論文は全4章で構成される。

第1章では、研究の背景としてPCB製造における露光技術の変遷とLDI技術の重要性を概説した。その上で、現行の355 nm光源が抱える技術的課題を整理し、本研究で提案するMgO:SLT導波路型QPMデバイスの優位性と研究目的、および論文の全体構成を述べた。

第2章では、非線形光学結晶を用いた波長変換の基礎理論について述べる。特に、MgO:SLTの材料特性（屈折率分散式など）や、基本波からSHGおよびSFGを経てTHGを得るための物理的プロセス、およびQPMの原理を述べる。

第3章では、本研究の目標達成において必要な、2$\mu$m 周期分極反転構造の実現に向けた作製プロセスの検討について述べる。MgO:SLT薄膜の接合や電子線（EB）描画を用いた電極形成、さらに高電界印加による分極反転の試作工程を詳述する。微細周期の形成において直面した技術的困難さや、試作結果から得られた知見に基づき、現在の到達点と構造実現に向けた課題を整理する。

第4章では、本研究で得られた成果を総括するとともに、実用化に向けた課題と今後の展望について述べる。
