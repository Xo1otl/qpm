# 結言

本章では、本研究の理論的基盤となる強誘電体結晶における波長変換理論について述べた。

はじめに、強誘電体結晶 MgO:SLT の材料特性について整理し、特に本研究で重要となる異常光線屈折率のセルマイヤー方程式を示した。
続いて、第二高調波発生 (SHG) および和周波発生 (SFG) のそれぞれについて、擬似位相整合 (QPM) を適用することで位相不整合を補償し、それぞれの変換効率を最大化できることを述べた。その際、所望の波長変換を実現するために必要な分極反転周期 $\Lambda$ を示した。

本章で示した理論および設計式は、次章における実際の $2 \mu\mathrm{m}$ 周期分極反転構造の作製プロセスの検討、およびその後のデバイス評価において基礎となるものである。次章では、この極めて短い周期構造を実現するための具体的な作製技術について詳述する。
