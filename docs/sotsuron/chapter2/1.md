# 緒言

本章では、本研究の基礎となる強誘電体非線形光学結晶を用いた波長変換の理論について述べる。
まず、非線形光学効果の基礎として、強誘電体結晶における非線形分極と非線形光学テンソルについて説明する。
次に、本研究で用いる強誘電体結晶である MgO:SLT の材料特性と、波長分散を表すセルマイヤー方程式を示す。
さらに、第二高調波発生 (SHG) および和周波発生 (SFG) における結合モード理論に基づいた結合波方程式を導出し、擬似位相整合 (QPM) デバイスにおいて変換効率を最大化するための分極反転周期の設計について述べる。
これらは、第3章以降で述べるデバイス作製および第4章での総括における基礎となるものである。
